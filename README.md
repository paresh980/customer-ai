# Smartnode Customer Support — AI/ML POC

## Project Structure

```
smartnode-customer-ai/
├── presentation.html                      # Deliverable 1: Use-case slide
├── Smartnode_Call_Transcription_POC.ipynb  # Deliverable 2: Jupyter/Colab notebook
├── smartnode_call_transcription_poc.py     # Deliverable 2 (alt): Standalone Python script
├── requirements.txt                       # Dependencies
├── sample_audio/                          # Generated audio files (auto-created)
└── results/                               # Output JSON + charts (auto-created)
```

## Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. View the presentation
Open `presentation.html` in any browser.

### 3. Run the POC

**Option A — Jupyter / Google Colab:**
Upload `Smartnode_Call_Transcription_POC.ipynb` to Google Colab and run all cells.

**Option B — Command line:**
```bash
python smartnode_call_transcription_poc.py
```
